# Estudos práticos de People Analytics utilizando PostgreSQL, SQL, Python, Pandas e Tableau para desenvolvimento de projetos de Analytics em RH e Departamento Pessoal.

Projeto de People Analytics para estudos voltados à remuneração utilizando PostgreSQL, SQL, Python, Pandas e Tableau.

## Objetivo

Criar indicadores eficientes para a tomada de decisão, apoiar a previsibilidade de impactos financeiros em relação a ajustes salariais, custo anual de folha e identificar os departamentos com os maiores custos salariais.

## Tecnologias utilizadas

* PostgreSQL
* DBeaver
* Python
* Pandas
* SQLAlchemy
* OpenPyXL
* Tableau

## Indicadores desenvolvidos

* Headcount
* Custo salarial por departamento.
* Custos por ajustes salariais
* Custo anual da folha.
* Custo por departamento.
* Custo de benefícios.

## Estrutura do projeto

* sql/
* exports/
* logs/
* dashboard/
* docs/
* images/
* src/
* data/

## Fluxo ETL

1. Extração dos dados do PostgreSQL
2. Leitura dos arquivos SQL
3. Processamento via Python/Pandas
4. Exportação para Excel
5. Visualização no Tableau
