# Estudos práticos de People Analytics utilizando PostgreSQL, SQL, Python e Tableau para desenvolvimento de projetos de Analytics em RH e Departamento Pessoal.

Projeto de People Analytics para estudos voltados à remuneração utilizando PostgreSQL, Python e Tableau.

## Objetivo

Criar indicadores eficientes para a tomada de decisão, apoiar a previsibilidade de impactos financeiros em relação a salário, custo anual de folha e identificar os departamentos e cargos com os maiores custos salariais.

## Tecnologias utilizadas

* PostgreSQL
* DBeaver
* Python
* Faker
* Pandas
* SQLAlchemy
* OpenPyXL
* Tableau
* GitHun

## Indicadores desenvolvidos

* Folha salarial
* Salário médio
* Custo total
* Diversidade
* Ranking 15 salários
* Headcount
* Senioridade
* Faixa etária
* Custo salarial por departamento
* Salário por departamento
* Custo por cargo
* Headcount por departamento
* Tempo de empresa por faixa etária
* Salário médio por departamento
* Tabela geral

## Estrutura do projeto

* assets/
* dashboard/
* logs/
* src/


## Fluxo ETL

1. Extração dos dados do PostgreSQL
2. Leitura dos arquivos SQL
3. Processamento via Python/Pandas
4. Exportação e Impostação dos dados através através de pipeline
5. Visualização no Tableau
